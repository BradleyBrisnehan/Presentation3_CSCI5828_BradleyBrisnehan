Boundary Scan Architecture

The JTAG standard defines both mandatory and optional instructions for interfacing with the boundary-scan layer. The following boundary-scan instructions are required for every JTAG / TAP implementation.

  • EXTEST
  • SAMPLE/PRELOAD
  • BYPASS

JTAG also defines several optional instructions. These instructions can be supported by the chip manufacturer to extend the functionality of the standard boundary-scan architecture. The follwoing optional instructions exist:

  • HIGHZ
  • CLAMP
